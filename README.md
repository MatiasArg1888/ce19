# CODIGO ESCONDIDO 19

Aplicacion Flet con version web publicada mediante GitHub Pages.

## Abrir en la web

Cuando GitHub termine de publicar la pagina, se abre desde:

https://matiasarg1888.github.io/ce19/

Si GitHub Pages no esta habilitado, entrar al repositorio y activar:

Settings > Pages > Build and deployment > Source: GitHub Actions

## Crear instaladores

Los accesos para crear Android y Windows estan en la carpeta `instalador`.
Abrir `instalador/CREAR_INSTALADOR.bat` para elegir el destino.
